Test if the server is affected by the Shellshock vulnerability.

== Description ==

Shellshock is the code name for a security bug found in Bash shell, used on UNIX systems.

The bug allows attacker to trick Bash into running a program that it wasn't supposed to. This has very serious security implications. [Read more](https://managewp.com/shellshock-wordpress-check "Shellshock WordPress plugin")

Shellshock plugin tests for both disclosed shellshock vulnerabilities CVE-2014-6271 and CVE-2014-7169.

Created by [ManageWP](https://managewp.com "ManageWP")

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder
2. Go to the Plugins page and activate the plugin
3. Visit Settings -> Shellshock and Run Test
